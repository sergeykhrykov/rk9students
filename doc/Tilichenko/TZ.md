# Техническое задание

## Разработка программного комплекса: "Web-viewer for 3D models"

### 1. Программный комплекс должен выполнять следующие функции:
1. Регистрация пользователей в системе с помощью электронной почты и пароля
2. Аутентификация в системе зарегистрированных пользователей
3. Загрузка 3D-моделей в форматах .stl, .anotherFormat и их просмотр
4. Хранение загруженных файлов
5. Выгрузка из хранилища ранее загруженных ранее пользователем данных

### 2. Требования к информационной и программной совместимости:
1. Клиентская часть приложения должна быть реализована на языке JavaScript
2. Серверная часть должна быть совместима с библиотеками, написанными на языке C++

### 3. Этапы разработки:
1. Разработка серверной части приложения, способного предоставлять пользователем html страницы
2. Разработка UI
3. Разработка БД
4. Разработка системы аутентификации
5. Разработка системы удаленного хранилища
